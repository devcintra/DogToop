Dog Top - O melhor para o seu Pet

Sistema Web com Banco de Dados para Petshops

Bem-vindo ao projeto Dog Top! Este sistema foi desenvolvido com o objetivo de criar uma plataforma intuitiva para o gerenciamento de petshops, unindo tecnologias Front-end (HTML, CSS e JS) e Banco de Dados.

Aqui você encontrará funcionalidades como o catálogo de produtos, carrinho de compras e um sistema prático para agendamento de banho e tosa. Nosso propósito é facilitar a vida dos donos de pets e a administração da loja!
